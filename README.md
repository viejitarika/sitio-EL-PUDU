# Sitio Web Pudú

Este proyecto fue desarrollado como parte de la evaluación de Programación Front End.

## Tecnologías utilizadas
- HTML5
- CSS3

## Uso de Inteligencia Artificial (IA)

Se utilizó inteligencia artificial como apoyo durante el desarrollo del proyecto.

### Prompts utilizados:

- "Guia de mejora en  el diseño CSS para que la página se vea más moderna"
- "Cómo alinear imágenes y texto usando flexbox en CSS"
- "Cómo modificar la galería de imágenes en HTML y CSS"
- "Cómo mejorar el diseño visual de una página web simple"
- "Cómo organizar un sitio web con múltiples páginas en HTML"

La inteligencia artificial fue utilizada únicamente como herramienta de apoyo.

## Características de CSS3 utilizadas

- Uso de Flexbox para la distribución de elementos
- Uso de border-radius para bordes redondeados
- Uso de box-shadow para efectos visuales
- Uso de hover para interacción en navegación
- Uso de background-image para diseño del encabezado


## Características de HTML utilizadas

- Uso de estructura semántica con etiquetas como <header>, <nav>, <main> y <footer>
- Uso de enlaces con la etiqueta <a> para navegación entre páginas
- Uso de imágenes con la etiqueta <img> para contenido visual
- Uso de listas (<ul> y <li>) para organizar información
- Uso de tablas (<table>, <tr>, <td>) para mostrar datos estructurados
- Uso de formularios básicos para la sección de contacto
- Uso de etiquetas de texto como <h1>, <h2>, <p> para jerarquía de contenido
- Uso de rutas relativas para organizar archivos en carpetas
